При разработке проекта использовал платформу ASP.NET, технологию Entity Framework и библиотеку js React.
В проекте две папки:
DrinksVendingMachine.Data - библиотека классов: модель базы данных с основными сущностями и db-контекстом
DrinksVendingMachine.Web.React - веб-приложение для покупки напитков c использованием React
  ./admin-page - доступ к странице администрирования данных

В корне проекта положил резервную копию базы данных VendingMAchine.bak
Ссылка на репозиторий проекта: https://github.com/silakovvv/drinks_vending_machine
